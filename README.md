Create .env.local file in the project root and add inside
1) `NEXT_PUBLIC_API_URL = http://127.0.0.1:8000`

3) `NEXT_PUBLIC_DOMAIN = 127.0.0.1`

4) `NEXT_PUBLIC_GOOGLE_ANALYTICS = G-8S8G1SC3LP`

Run the client: 
`npm install` `npm run dev`
